# 💚 HealthWise AI - 1M1B Internship Project

## 🎓 Project Overview

**HealthWise AI** is a preventive healthcare companion developed as part of the **1M1B + IBM SkillsBuild + AICTE AI for Sustainability Internship Program**.

### 🎯 Objective
Leverage AI technology to promote preventive healthcare and contribute to **UN SDG 3: Good Health and Well-being**.

---

## 🚀 Quick Start

### Installation

```bash
# Install dependencies
python -m pip install -r requirements.txt

# Run the application
streamlit run app.py
```

### Access
Open your browser at: **http://localhost:8501**

## 📊 Key Metrics

### Wellness Score Algorithm
```
Total Score (0-100) = 
  Sleep Score (25 points) +
  Hydration Score (20 points) +
  Exercise Score (20 points) +
  Energy Score (15 points) +
  Stress Score (20 points)
```

### Categories
- **Excellent**: 85-100
- **Good**: 70-84
- **Needs Attention**: 50-69
- **High Risk**: <50
